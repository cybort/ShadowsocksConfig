## Development

First [install yarn](https://yarnpkg.com/en/docs/install-ci).

The Gulpfile itself is written in TypeScript.
You can run the package script `gulp` to compile and run it on the fly:

```sh
yarn run gulp
```

### Unit Tests

```sh
yarn run test
```
